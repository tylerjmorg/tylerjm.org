Record Inventory
Definitions

Qty
Quantity (Qty) of the exact same record in the inventory. For records with a Qty above 1, all records must be identical to one another and must have the same properties (e.g., Disc Color(s), Disc Count, Contents Count, etc.). A 0 Qty indicates that a record once belonged in the inventory but has changed possession willfully (e.g., sold or gifted).
Disc Color(s)
Signifies different variants and styles of discs within a single record (e.g., Target Exclusive (Blue) or simply Blue).
Disc Count
The quantity of original discs contained within a single record.
Contents Count
All contents in the originally purchased record, including any original sleeves, jackets, digital download slips, posters, booklets, hype stickers, and other bonus content. Post-sale sleeves or any other aftermarket items will not be counted. The contents count excludes discs.
Contents Description
The quantity and brief description of all original contents contained within one record, including any original sleeves, jackets, digital download slips, posters, booklets, hype stickers, and other bonus content. Post-sale sleeves, discs, and any aftermarket items are exlcuded. The total quantity of the Contents Description must be equal to the Contents Count property.
Discogs Release Code
Identifies the specific record at https://www.discogs.com. Searching the release code provides extensive information regarding a record. An example of a Discogs release code is r26128271.
Goldmine® Condition(s)
Each disc and jacket's condition according to The Goldmine® Grading Guide. URL: https://www.goldminemag.com/collector-resources/record-grading-101. Each disc and jacket must be graded individually. Acceptable grades:
  • M (Mint) Very rarely used
  • NM (Near Mint)
  • VG+ (Very Good Plus)
  • VG (Very Good)
  • G+ (Good Plus)
  • G (Good)
  • F (Fair)
  • P (Poor)
Listened
A Boolean value identifying if all tracks on a specific record have been played and listened to in its entirety. All quantities must be listened to receive a YES value; i.e., if the Qty column is 3, all three records must be listened to in their entirety, even though they are the same tracks.
Digital Download(s)
A Yes, No, or N/A value indicating if a digital format of the exact same record was downloaded. The digital download must include the same album name and the same and/or more tracks as the physical record (excluding physical format exlcusive tracks). No value means that there exists a buyable download that meets the Yes value requirements but has not been purchased. N/A value means that there is no existing buyable download that meets the Yes value requirements. Yes value requirements:
  • The download must contain at least an ALAC, FLAC, or WAV original file for all tracks.
  • The download has to be an official download from the label, artist, or an authorized 3rd party (e.g., Qobuz).
  • The download must be a portable file (i.e., not downloaded from iTunes or another proprietary store).
  • The proof of purchase (PoP) (digital download slips are okay) must be stored safely alongside the music file(s) in cloud storage.
